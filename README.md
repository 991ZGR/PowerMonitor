# PowerMonitor

整机功耗监控工具，支持 CPU（RAPL）和 GPU（NVML）实时功耗显示，屏幕右上角叠加层，系统托盘运行。

## 下载

[GitHub Releases](https://github.com/991ZGR/PowerMonitor/releases) 页面下载 `PowerMonitor.exe`。

> 需安装 .NET 8.0 Desktop Runtime。首次运行会自动安装 PawnIO 内核驱动（弹窗确认即可），用于读取 CPU MSR 寄存器。

## 使用

**首次运行必须右键 → 以管理员身份运行**。

| 操作 | 说明 |
|------|------|
| 拖拽标题栏 | 移动窗口 |
| 点击 [⚡] | 切换刷新速度 |
| 点击 [P] | 切换悬浮窗（右上角叠加层） |
| 点击 [⚙] | 悬浮窗设置（位置、大小、透明度） |
| 点击 [—] | 最小化 |
| 点击 [✕] | 最小化到系统托盘 |
| 双击托盘图标 | 恢复窗口 |
| 右键托盘图标 | 显示窗口 / 退出 |

## 悬浮窗

点击 **P** 开启右上角悬浮窗，类似游戏准星——不可点击穿透不抢焦点。

点击 **⚙** 可调节：
- 左右位置 / 上下位置
- 缩放大小
- 不透明度

## 刷新速度

点击 **⚡** 切换五档速度：
- 🔥 疯狂 10ms / 🚀 极速 100ms / ⚡ 快 500ms / ⏱ 中 1000ms / 🐢 慢 2000ms
